# Getting started

Clone our repository:
$ git clone https://github.com/imgox/RT_final
Make sure that you have OpenGL installed. and make:
$ make
after successfully compiling the objects you can run RT like this:
$ ./rt [scene-file]

# Scene file

We accept scene file in XML format with the following requirements and conditions:

# World object
Every scene file should have a world object.
example:
<world ambient="20%" />
# Screen object
Every scene file should have a screen object.
example:
<screen resolution="25%" />
# Camera object
Every scene file should have a camera object.
example:
<camera position="(0, 0, 50)" lookat="(0, 0, 0)" />

# Interface

Our very friendly user interface that has made our asses sweat with SEGV errors. Let's walk you through our major buttons.
Add sphere

This button let you add a sphere to the scene in-program.

Add cylinder

This button let you add a cylinder to the scene in-program.

Add cone

This button let you add a cone to the scene in-program.

Add plane

This button let you add a plane to the scene in-program.

Voronoi noise

This button let you change the noise of the selected object to the voronoi noise in-program.

Perlin noise

This button let you change the noise of the selected object to the perlin noise in-program.

Damier noise

This button let you change the noise of the selected object to the damier noise in-program.

There's also a handy color picker to change the color of the selected object to pretty much any color you want.

# User experience

here's a little guide to use our program:
Using your keyboard:
Button Use
+ Increment resolution
- Decrement resolution
→ Move the selected object in the positive x axis
← Move the selected object in the negative x axis
↑ Move the selected object in the positive y axis
↓ Move the selected object in the negative y axis
W Move the selected object in the positive z axis
S Move the selected object in the negative z axis
Esc Quit the program
Using your mouse:

Right button: Select an object

Left button: Deselect, interact with the UI elements
